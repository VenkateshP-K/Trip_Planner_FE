# React + Vite

This template provides a minimal setup to get React working in Vite.

## Getting Started

Run `npm install` to install dependencies.

Then run `npm run dev` to start the development server.

## technologies

- React-vite
- bootstrap

## Features

- user register
- user login
- user dashboard
- user logout
- add trips
- edit trips
- delete trips
- manage travel
- manage accommodation
